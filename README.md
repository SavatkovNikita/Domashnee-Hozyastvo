## PDF_NNTU_Parser
## О проекте
Проект представляет собой система, учитывающую количество продуктов в холодильнике, список продуктов, количество бытовой химии в шкафу, и платёжки. Цель системы - контроль запасов продуктов и бытовой химии, упрощение процесса готовки за счёт знания имеющихся ингридиентов, консолидация финансовых потоков в едином интерфейсе. Рыбота сделана в Python
## Функции
1.	Учет продуктов в холодильнике
2.	Учет бытовой химии
3.	Учет платежек
## Пользователь
1.	может просматривать все хранилище
2.	может изменять продукты в холодильнике
3.	может платить за платежки + просмотр расходов (вывод отчета)
4.	изменять содержимое шкафа
5.	может оплачивать платежки
6.	может получать зарплату


## Принцип работы парсера
Програмы для шкафа, холодильника, оплаты платёжек представляют собой функцию с циклом и условиями.
В программе для холодильника мы вводим, обём холодильника, название продукта, объём этого продукта, и его количество, аналогично для шкафа. Для платёжек мы вводим зарплату, платёжку и сколько мы ходим заплатить.
## Что нужно для запуска?
Данные о бытовой химии, продуктах, холодильнике, шкафе, заработной плате, платёжках
