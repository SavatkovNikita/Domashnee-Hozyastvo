##PDF_NNTU_Parser
##О проекте
Проект представляет собой сервис, в котором пользователь может посмотреть и изменить список продуктов и бытовой химии, и заплатить платёжки. Рыбота сделана в Python
##Принцип работы парсера
Програмы для шкафа и холодильника представляют собой функцию с циклом и условиями, в программе для выплаты платёжек только функция с условием
##Что нужно для запуска?
Python
